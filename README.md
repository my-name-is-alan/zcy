# ZCY

使用 `QWik`构建的SSR 网站

## 使用方法

```bash
git clone xxxx
cd zcy
npm install
pnpm start
```

## Express Server

This app has a minimal [Express server](https://expressjs.com/) implementation. After running a full build, you can preview the build using the command:

```
pnpm serve
```

Then visit [http://localhost:8080/](http://localhost:8080/)
